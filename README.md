# Sales
# European E-Commerce Sales & Revenue Dashboard (2019 - 2020)

## 📌 Project Overview
This repository contains a comprehensive **Sales and Revenue Analysis Dashboard** built to track, analyze, and visualize retail transaction metrics across multiple European markets. The project bridges raw transaction logs with stakeholder-ready visual insights to evaluate country-level performance, platform (device) usage behavior, and product category popularity.

---

## 📊 Dashboard Visualizations & Insights

The analytical frontend consists of four main reporting modules:

1. **Sales Distribution by Device Type (Pie Chart)**
   * **Insight:** PC users account for the overwhelming majority of revenue ($89.6M), outperforming Mobile ($15.4M) and Tablet ($8.2M) combined.

2. **Sales by Product Category (Bar Chart)**
   * **Insight:** *Clothing* dominates as the top-grossing category (generating over $17.6M), closely followed by *Games* and *Appliances*.

3. **Monthly Sales Trend Line (2019 - 2020)**
   * **Insight:** Captures cyclical transaction volatility over a 24-month horizon, mapping clear seasonal spikes and revenue valleys.

4. **Total Sales and Profit by Country (Stacked Horizontal Bar Chart)**
   * **Insight:** Identifies the highest-performing international hubs, revealing that the **UK** and **Netherlands** serve as primary revenue engines.

---

## 📁 Repository Contents

* `/data`: Contains the foundational `Sales Data` file featuring 1,000 unique transaction records tracking `order_id`, `country`, `order_value_EUR`, `cost`, `category`, `sales_manager`, and `device_type`.
* `/dashboard`: Contains the spreadsheet architecture housing the primary data sheet, backend pivot tables, and visual reporting charts.
